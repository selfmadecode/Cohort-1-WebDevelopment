What is CSS?

CSS (Cascading Style Sheets) is the language used to style and layout web pages ‚Äî changing colors, spacing, fonts, or positioning elements. It works with HTML to make web pages look attractive.

CSS Syntax

selector {
  property: value;
}

Selector: Targets the HTML element you want to style (e.g., h1, .class, #id)

Property: The style feature to change (e.g., color, font-size)

Value: The setting you want to apply (e.g., red, 20px)

p {
  color: blue;
  font-size: 18px;
}

CSS Margin vs Padding

Margin

Space outside an element.

p {
  margin-top: 20px;
}

Padding

Space inside an element.

p {
  padding-top: 20px;
}

Negative Values

  • Negative Margin: Allowed; moves elements closer or overlaps them.
    • p { margin-top: -5px; }
  • Negative Padding: ‚ùå Not allowed.

CSS Height and Width

div {
  height: 100px;
  width: 100%;
}

Tip: Use units like px, %, vh, or vw to control layout size.

Responsive Design with Media Queries

Media queries help web pages adjust to different screen sizes (mobile, tablet, desktop).

@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

When the screen is 600px or smaller, the background becomes light blue.

@media (max-width: 768px) { ... }  /* Tablets */
@media (max-width: 576px) { ... }  /* Mobile */
